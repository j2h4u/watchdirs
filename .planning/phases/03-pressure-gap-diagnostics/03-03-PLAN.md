## Contract

`docker-enrichment` normalizes Docker CLI category evidence only from fixed read-only Docker commands. It must not claim containerd category totals from `/var/lib/containerd` paths. If persisted indexed paths or explicit path hints include `/var/lib/containerd` and no containerd-native probe result is present, the output must include `containerd_available=false`, `containerd_path_hints`, and a warning with code `containerd_enrichment_unavailable`. Containerd path hints are evidence that containerd storage may matter, not reclaimable/active category evidence.
