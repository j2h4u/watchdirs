## Contract

`df-vs-index` success JSON must include `ok`, `command`, `snapshot_selector`, `limit`, `effective_limit`, `filesystems`, `summary`, and `warnings`. Each filesystem section includes storage-domain label, selected snapshot ids, snapshot finished-at min/max, maximum snapshot age seconds, `df_bytes.size`, `df_bytes.used`, `df_bytes.free_total`, `df_bytes.avail_unprivileged`, `indexed_visible_disk_bytes`, `indexed_visible_path_count`, partial/unknown counters, `unattributed_bytes`, `unattributed_ratio`, `over_indexed_bytes`, `likely_reasons`, `verification_commands`, and `truncated`.

Material mismatch defaults: `MISMATCH_MIN_BYTES = 1 * 1024 ** 3` and `MISMATCH_MIN_RATIO = 0.05`. A mismatch is material when both the byte floor and ratio threshold are met. Ratios use filesystem used bytes as the denominator when positive.
