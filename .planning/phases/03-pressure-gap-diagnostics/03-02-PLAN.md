| Boundary | Description |

| CLI to process table | The diagnostic reads live process/fd state that may be permission-limited. |
| External command output to parser | `lsof` output can be malformed, partial, or accompanied by warnings. |
| Procfs links to renderer | Process names and paths can contain confusing text. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-06 | Elevation of Privilege | lsof/procfs probe | mitigate | Do not escalate privileges; surface permission gaps as warnings and counters. |
| T-03-07 | Tampering | subprocess invocation | mitigate | Use fixed argv, `shell=False`, no user command interpolation. |
| T-03-08 | Spoofing | process/path text rendering | mitigate | Escape text output and keep JSON fields explicit. |
| T-03-09 | Denial of Service | culprit volume | mitigate | Apply strict limit and truncation metadata. |
