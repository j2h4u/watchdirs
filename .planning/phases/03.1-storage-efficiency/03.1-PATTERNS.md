### `src/watchdirs/db/schema.sql` (schema, CRUD)

**Analog:** itself, current lines 11-44 (the `directory_sizes` table + 3 path-bearing indexes).

**Current table to replace** (lines 11-23): inline `path BLOB`, `parent_path BLOB`, `name BLOB`.

**New shape (D-01/D-02/D-03):** add a `paths` table; `directory_sizes` carries `path_id`/`parent_id` ints; drop `name`. Note Research Pitfall 2 — declare `path TEXT` but the column will store `bytes` losslessly (`typeof=blob`); declaring TEXT is intent only.

```sql
CREATE TABLE IF NOT EXISTS paths (
    id INTEGER PRIMARY KEY,
    path TEXT NOT NULL UNIQUE          -- bytes bound here; UNIQUE creates sqlite_autoindex_paths_1
);

CREATE TABLE IF NOT EXISTS directory_sizes (
    id INTEGER PRIMARY KEY,
    snapshot_id INTEGER NOT NULL REFERENCES snapshots(id) ON DELETE CASCADE,
    path_id INTEGER NOT NULL REFERENCES paths(id),
    parent_id INTEGER REFERENCES paths(id),   -- nullable, preserves diff COALESCE (D-03)
    depth INTEGER NOT NULL,
    apparent_bytes INTEGER NOT NULL,
    disk_bytes INTEGER NOT NULL,
    file_count INTEGER NOT NULL,
    dir_count INTEGER NOT NULL,
    error TEXT
);
```

**Index int-ization** (replaces current lines 37-44 — this is a large share of the win, D-03):
```sql
CREATE INDEX IF NOT EXISTS directory_sizes_pathid_snapshot_idx ON directory_sizes(path_id, snapshot_id);
CREATE INDEX IF NOT EXISTS directory_sizes_snapshot_size_idx   ON directory_sizes(snapshot_id, disk_bytes);
CREATE INDEX IF NOT EXISTS directory_sizes_snapshot_parent_idx ON directory_sizes(snapshot_id, parent_id);
```

### `src/watchdirs/db/migrations.py::_resolve_path_id` (NEW utility — no exact analog)

**Nearest structural pattern:** the value-build + cache discipline around `_directory_row_values`; canonical form is Research Pattern 1 (verified on host sqlite 3.50.4). NOT `INSERT OR IGNORE ... RETURNING` (Pitfall 1 — returns empty on the dominant duplicate case).

```python
def _resolve_path_id(connection, cache: dict[bytes, int], path: bytes) -> int:
    cached = cache.get(path)
    if cached is not None:
        return cached
    row = connection.execute("SELECT id FROM paths WHERE path = ?", (path,)).fetchone()
    if row is not None:
        path_id = int(row[0])
    else:
        cur = connection.execute("INSERT INTO paths(path) VALUES (?)", (path,))
        path_id = int(cur.lastrowid)
    cache[path] = path_id
    return path_id
```
- Bind raw `bytes`, never a decoded `str` (Pitfall 2 — surrogateescape `str` raises `UnicodeEncodeError`).
- `parent_path is None` → `parent_id` stays `None` (mirrors current line 208 `... if row.parent_path is not None else None`).
- Cache lives for the duration of one `insert_directory_rows` call; hit-rate ~100% after snapshot #1 (D-04).

## Shared Patterns

### Raw-bytes binding (load-bearing — Phase 01 decision, Pitfall 2)
**Source:** `src/watchdirs/db/migrations.py:207` (`sqlite3.Binary(row.path)`), `reporting/queries.py:142` (`bytes(row["path"])`).
**Apply to:** `_resolve_path_id`, every `JOIN paths` SELECT, all fixtures. Bind `bytes`, never decoded `str`. Reads return `bytes(row["path"])`.

### Parameterized queries only (ASVS V5 — Research Security Domain)
**Source:** every `connection.execute(..., (params,))` in the codebase; docker GLOB at `cli.py:920` binds `(snapshot.id, prefix, prefix + b"/*")`.
**Apply to:** new dedup INSERT/SELECT and the new `JOIN paths` GLOB. Never interpolate path bytes.

### Schema rebuild, no migration shim (project rule)
**Source:** `migrations.py:15-37` `initialize_database` — `IF NOT EXISTS` + `PRAGMA user_version`, raises if on-disk is newer. NO down-migration.
**Apply to:** bump `SCHEMA_VERSION` to 3, ship fresh `schema.sql`. Dev DBs regenerate. Do not add any compat/fallback path.

### stdout = JSON / stderr = everything-else (D-11)
**Source:** `cli.py:977 emit_json` (the only stdout writer for JSON), `cli.py:986,1010 print(..., file=sys.stderr)`.
**Apply to:** all new `logging` observability — StreamHandler on `sys.stderr` only.

### Batched executemany ingestion (reusable asset)
**Source:** `migrations.py:93-95` `INSERT_BATCH_SIZE` loop + `executemany`.
**Apply to:** the rewritten `insert_directory_rows` and the benchmark's synthetic-snapshot replication in `bench/size.py`.
