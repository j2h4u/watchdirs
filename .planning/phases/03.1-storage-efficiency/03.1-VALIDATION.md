## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | pytest (NOT a declared dep; injected via `uv run --with pytest`) |
| **Config file** | `pyproject.toml` `[tool.pytest.ini_options]` (`pythonpath=["src"]`, `testpaths=["tests"]`) |
| **Quick run command** | `uv run --with pytest pytest -q -x` |
| **Full suite command** | `uv run --with pytest pytest -q` |
| **Estimated runtime** | ~10–30 seconds (unit-heavy; benchmark harness runs separately, not in the suite gate) |

---

## Sampling Rate

- **After every task commit:** Run `uv run --with pytest pytest -q -x`
- **After every plan wave:** Run `uv run --with pytest pytest -q`
- **Before `/gsd-verify-work`:** Full suite must be green **AND** benchmark PASS (per-snapshot byte budget, D-09)
- **Max feedback latency:** ~30 seconds (suite); benchmark is a separate phase-gate artifact, not in the per-commit loop
